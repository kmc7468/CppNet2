# CppNet2
.NET Standard Implementation for C++17

CppNet2는 [CppNet](https://github.com/kmc7468/CppNet)의 후속작으로, .NET Standard의 네이티브 구현체입니다. .NET Standard를 있는 그대로 옮기면서도 C++의 실정에 맞게 일부 개선했습니다.
